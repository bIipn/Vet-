# Vet Clinic Manager

**A console-based C# application for managing animal records in a veterinary clinic.**

Allows users to add, view, update, and list animals with details like species, ID, age, description, personality, and nickname. Built to practice array/list management, user input handling, and menu-driven console apps in .NET.

## Features
- Add new animal records
- List all animals with formatted output
- Search/view details by ID
- Basic validation (age, required fields)
- Simple text-based menu interface

## Tech Stack
- **C#** (.NET 6+ / .NET 8+)
- Console Application
- Collections (arrays or List<T>)
